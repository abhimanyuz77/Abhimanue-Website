💻 COURSE PLAN

🎯 Course Goal

By the end of 12 weeks, students will:

  • Understand programming and computing fundamentals
  • Build and host a complete web application
  • Use tools like Git, Firebase, and basic React
  • Learn how modern developers work with AI, automation, and the cloud
01

Week 1: The World of Programming & Computing

Theme: Fundamentals, Industry Landscape, and AI in Development

  • Topics Covered

    How the web works (frontend, backend, databases, APIs) • Understanding programming logic, algorithms, and problem-solving • Data structures: arrays, lists, loops, conditions (in JS) • How AI and automation are reshaping coding

  • Mini Project

    Write a small algorithm to reverse a string or sort numbers

  • Outcome

    Students understand what coding actually is and how developers think

02

Week 2: Web Foundations — HTML & CSS

Theme: Building and Styling Webpages

  • Topics Covered

    HTML structure, tags, forms, semantic elements • CSS basics: colors, layouts, flexbox, responsive design • Figma intro — how to visualize designs • Hosting static sites using Cloudflare Pages, GitHub Pages, or Netlify

  • Mini Project

    Personal "About Me" website deployed online

  • Outcome

    Students see their first live website

03

Week 3–4: JavaScript Fundamentals

Theme: Programming for the Web

  • Topics Covered

    JS basics: variables, arrays, objects, loops, functions • DOM manipulation — how to make pages interactive • Handling user input and forms • Browser console and debugging

  • Mini Project

    "To-Do List" or "Calculator" web app

  • Outcome

    Students create an interactive web page with JS

04

Week 5–6: Intermediate JavaScript + Modern Tools

Theme: Smart Coding Practices

  • Topics Covered

    Event handling, JSON, local storage • APIs: Fetching data from external APIs • ES6+ concepts (arrow functions, promises, async/await) • Introduction to React (concepts, components, JSX)

  • Mini Project

    "Weather App" using a free weather API

  • Outcome

    Students fetch data from real-world APIs

05

Week 7: Version Control & Collaboration

Theme: Working Like a Developer

  • Topics Covered

    Git and GitHub basics: cloning, committing, branching, pull requests • How open-source collaboration works • Markdown, documentation, and README creation

  • Mini Project

    Push their "Weather App" to GitHub and deploy it again

  • Outcome

    Students can use GitHub like pros

06

Week 8–9: React Essentials

Theme: Building Modern Frontends

  • Topics Covered

    React component structure and props • State management and hooks • Building a small React project from scratch

  • Mini Project

    "Notes App" or "Expense Tracker" in React

  • Outcome

    Students understand core React workflows

07

Week 10: Backend & Databases (Intro with Firebase)

Theme: Full Stack App Setup

  • Topics Covered

    What is a backend and how Firebase makes it simple • Firestore database basics: CRUD operations • Authentication (login/signup with Google)

  • Mini Project

    Add login and database to their React app

  • Outcome

    Students connect a database and auth system

08

Week 11: Bringing It All Together

Theme: Build Your Final App

  • Topics Covered

    Full-stack project planning and setup • Integrating frontend + backend + hosting • Adding small touches: favicon, metadata, responsive UI

  • Project

    Students start building their final app (choose theme: portfolio, note app, blog, or shop)

09

Week 12: Launch Week

Theme: From Developer to Creator

  • Topics Covered

    Deploying full app (Firebase Hosting / Vercel) • Writing documentation & demo video • Resume and LinkedIn optimization for developers

  • Final Project

    Launch their own app live and share it publicly

  • Outcome

    Receive certificate + GitHub portfolio + live project

🧠 Add-Ons / Bonus Modules (Optional)

  • AI for Developers: Using ChatGPT for debugging & brainstorming features
  • Intro to TypeScript: Why big companies use it
  • Next.js teaser: What's beyond React
  • Freelancing Pathway: How to use their new skills to earn money online
